# Dungeon Dash (Terminal)

A tiny turn-based dungeon crawl in your terminal. Collect treasure, dodge traps, outrun the shadow, then escape.

## Run

```bash
python3 terminal/dungeon_dash.py
```

## Show scores

```bash
python3 terminal/dungeon_dash.py --scores
```

## Optional seed

```bash
python3 terminal/dungeon_dash.py --seed 123
```

## Controls

- `W/A/S/D` to move
- `R` to rest (shadow still moves)
- `Q` to quit

## Goal

Collect at least 5 treasure and reach the `>` exit.
